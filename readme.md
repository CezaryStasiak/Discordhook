# Discordhook
As simple, as You want it.
```java
new Discordhook(url).toss(message);
```
or
```java
final Discordhook hookInstance = new Discordhook(url);
        discordHook.addMessage(message);
        discordHook.asUser(userName);
        discordHook.withAvatar(avatarUrl);
        discordHook.send();
```
